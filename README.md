# ПЗМ Завод Комплектации - Система категорий товаров

## Установка:

1. **Скопировать файлы в папки проекта:**

   - `js/product-data.js` → `/js/`
   - `js/product-categories.js` → `/js/`
   - `css/product-categories.css` → `/css/`

2. **Обновить каждый из 6 HTML файлов товаров:**

   **В секции <head> добавить:**

   ```html
   <link rel="stylesheet" href="../css/product-categories.css" />
   ```
